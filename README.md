# Printing Facility Inventory Management System

A web-based inventory management solution built from real-world experience in the printing industry. In printing facilities, we use a high volume of supplies daily—toner cartridges, paper rolls, machine oil, spare parts, and cleaning materials. Keeping track of these items effectively is critical to maintaining production uptime and avoiding costly delays.

This application provides real-time inventory tracking, automated low-stock alerts, and one-click usage logging to streamline material management operations.

---

## 🎯 Key Features

- **Real-Time Inventory Dashboard** - Visual overview with color-coded stock indicators and progress bars
- **One-Click Usage Tracking** - AJAX-powered quick logging without page reloads
- **Automated Low-Stock Alerts** - Notification badge and dedicated alert page for threshold monitoring
- **Quick Add System** - Category-specific modals with pre-populated fields for faster data entry
- **Search & Filter** - Real-time search across items with category filtering
- **CSV Import/Export** - Bulk operations for inventory management and data backup
- **Usage History Tracking** - Complete audit trail of all inventory changes with timestamps
